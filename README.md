# 🏡 Staylio — Vacation Rental Web Application

Staylio is a full-stack web application that enables users to browse, list, and book vacation accommodations such as houses, villas, and farmhouses. The platform provides a seamless booking experience, secure authentication, AI assistance, and real-time property location viewing.

---

## ✨ Features

- 🔎 Search and filter vacation properties
- 🏠 List your own property for rent
- 🔐 Secure authentication (JWT, Google OAuth)
- 💳 Razorpay payment integration for bookings
- ⭐ User reviews and ratings
- 💬 Messaging between users and property owners
- 📱 Fully responsive UI

---
